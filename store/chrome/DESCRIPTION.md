## Full Description

### Security Search - Your Shield Against DeFi Scams

Stop falling victim to scam sites in your search results. Security Search automatically warns you about known scam sites and verifies legitimate DeFi protocols.

### Key Features

**🛡️ Real-Time Protection**
- Automatically detects scam sites in Google, Bing, DuckDuckGo, Yahoo, and Baidu search results
- Shows warnings before you visit dangerous sites
- Verifies safe DeFi protocol sites

**📊 Pre-Built Database**
- 57+ verified DeFi protocol sites (Uniswap, Aave, Lido, Curve, Compound, and more)
- 29+ known scam sites already blocked
- Syncs with MetaMask and ScamSniffer blocklists

**🔄 Community-Driven**
- Report new scam sites you encounter
- Mark legitimate sites as safe
- Sync with community blocklists

**🎨 Beautiful UI**
- Dark and light theme support
- Hover tooltips with safety info
- Quick actions from any page

### Supported Protocols

**DEXs:** Uniswap, Curve, Balancer, SushiSwap, PancakeSwap, Orca, Raydium
**Lending:** Aave, Compound, Morpho
**Liquid Staking:** Lido, Rocket Pool
**Stablecoins:** MakerDAO, Frax
**Wallets:** MetaMask, Phantom, Rabby
**And many more...**

### Permissions

- **activeTab**: To check the current page
- **storage**: To save your scam/safe lists
- **Host access**: Only on search engines to scan results

### Privacy

- All data stored locally
- No tracking or analytics
- Open source under MIT license
